# JSONote Mini

JSONote Mini is a lightweight, dark-themed markdown note-taking application built with Next.js. It integrates directly with your GitHub repository, allowing you to create, edit, and save notes using a Personal Access Token.

## Features

- **GitHub Integration**: Direct read/write access to your repository using Octokit.
- **Markdown Support**: Create and edit notes in Markdown format.
- **Samsung Notes Aesthetic**: A premium, gray-scale dark mode interface inspired by Samsung Notes.
- **Local First**: Your GitHub token is stored securely in your browser's local storage.
- **Glassmorphism UI**: Modern, sleek design with glassmorphism effects.

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Run the development server: `npm run dev`
4.  Open [http://localhost:3000](http://localhost:3000) in your browser.
5.  Click the settings icon and enter your GitHub Personal Access Token and repository details.

# JSONote Mini

JSONote Mini는 Next.js로 구축된 가볍고 어두운 테마의 마크다운 노트 필기 애플리케이션입니다. GitHub 저장소와 직접 통합되어 Personal Access Token을 사용하여 노트를 생성, 편집 및 저장할 수 있습니다.

## 주요 기능

- **GitHub 통합**: Octokit을 사용하여 저장소에 직접 읽기/쓰기 액세스.
- **마크다운 지원**: 마크다운 형식으로 노트 생성 및 편집.
- **Samsung Notes 미학**: Samsung Notes에서 영감을 받은 프리미엄 회색조 다크 모드 인터페이스.
- **로컬 우선**: GitHub 토큰은 브라우저의 로컬 저장소에 안전하게 저장됩니다.
- **글래스모피즘 UI**: 글래스모피즘 효과가 있는 현대적이고 세련된 디자인.

## 시작하기

1.  저장소를 클론합니다.
2.  의존성을 설치합니다: `npm install`
3.  개발 서버를 실행합니다: `npm run dev`
4.  브라우저에서 [http://localhost:3000](http://localhost:3000)을 엽니다.
5.  설정 아이콘을 클릭하고 GitHub Personal Access Token 및 저장소 세부 정보를 입력합니다.
